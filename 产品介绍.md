## 📦 功能介绍

### 一、文件解压

支持 **8 种压缩格式** 的解压：

| 格式 | 说明 |
|------|------|
| .zip | 最通用的压缩格式 |
| .rar | WinRAR 格式 |
| .7z | 高压缩率格式 |
| .tar | Linux 归档格式 |
| .tar.gz / .tgz | 常见于开源软件 |
| .bz2 | 高压缩率 |
| .xz | 超高压缩率 |

**特色功能：**
- ✅ 拖拽文件到窗口即可解压
- ✅ 解压前预览压缩包内容
- ✅ 支持加密压缩包（密码解压）
- ✅ 自动创建同名文件夹，文件不散乱

### 二、文件压缩

**3 种输出格式：**

| 格式 | 特点 |
|------|------|
| .zip | 最通用，Windows 原生支持 |
| .7z | 压缩率更高 |
| .tar.gz | Linux 友好 |

**特色功能：**
- ✅ 支持加密压缩（zip / 7z）
- ✅ 可选择多个文件和文件夹
- ✅ 自动生成文件名

### 三、格式转换

#### 🖼 图片转换（10 种格式互转）

支持格式：**JPG、PNG、WebP、BMP、GIF、TIFF、ICO、SVG**

- 批量转换（一次选一堆图片）
- 可调节输出质量（10%-100%）
- 无需额外工具

#### 🎵 音频转换（7 种格式互转）

支持格式：**MP3、WAV、FLAC、AAC、OGG、M4A、WMA**

- 可选比特率（128k ~ 320k）
- 无损格式支持（FLAC、WAV）

#### 🎬 视频转换（7 种格式互转）

支持格式：**MP4、AVI、MKV、MOV、WMV、WebM、FLV**

- 可调节视频码率
- 纯本地处理，不上传任何文件

#### 📄 PDF 转换

- **图片 → PDF**：多张图片合成一个 PDF
- **PDF → 图片**：PDF 每页导出为图片
- 可调节 DPI（清晰度）

## ❓ 常见问题

**Q：解压时提示密码错误？**  
A：确认压缩包是否有密码保护，在"解压选项"中输入正确密码。

**Q：音视频转换失败？**  
A：软件内置了 ffmpeg 引擎，一般不会出现此问题。如遇特殊格式，请确认源文件未损坏。

**Q：转换后的文件在哪里？**  
A：默认保存在原文件所在目录，你也可以手动指定输出位置。

**Q：可以批量处理吗？**  
A：图片转换支持批量（一次选择多张），音频和视频目前单次处理一个文件。

**Q：会收集我的文件或隐私吗？**  
A：**不会。** 所有文件处理完全在本地进行，软件仅用机器码验证授权，不上传任何文件或个人数据。
